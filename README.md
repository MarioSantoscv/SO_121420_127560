### 1. Linux Recycle Bin System
A bash-based Linux Recycle Bin system implemented for the discipline "Sistemas Operativos." This project allows users to safely delete files with the ability to restore them before 
permanent deletion. The system mimics a desktop recycle bin but is accessible via shell commands, providing logging, statistics, auto-cleanup, and flexible configuration.
